## UI PROTECTION POLICY

The current visual identity is considered production-approved.

DO NOT redesign:
- Dashboard
- Landing Page
- Navbar
- Sidebar
- Cards
- Hero Sections
- Footer
- Color Palette
- Typography
- Component Styles
- Responsive Layout

The UI is LOCKED.

Only functional additions are allowed.

Examples of allowed changes:
✓ Add button
✓ Add modal
✓ Add form field
✓ Add new page using existing layout
✓ Add notification badge
✓ Add table column
✓ Add menu option

Examples of forbidden changes:
✗ Redesign dashboard
✗ Change colors
✗ Change typography
✗ Replace cards
✗ Move navigation
✗ Modify spacing
✗ Change visual identity
✗ Replace existing layout

If a visual modification is absolutely necessary, the AI must explain why and wait for user approval before implementing it.
